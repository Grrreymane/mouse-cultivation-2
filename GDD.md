## 1. 游戏概览

### 1.1 核心概念
一只鼠鼠踏上修仙之路，从炼气期凡鼠一步步修炼成仙。游戏自动战斗，玩家通过策略选择（装备、功法、灵兽、丹药）和资源管理推动进度。关闭浏览器后仍能积累收益（离线挂机）。

### 1.2 设计理念
- **零门槛启动**：打开即玩，无需注册/下载
- **有意义的挂机**：离线补算经验/灵石/材料，每次打开都有惊喜
- **多维养成**：装备 × 功法 × 灵兽 × 丹药 × 洞府 × 转生，避免单一升级路线
- **循环驱动力**：飞升转生机制让每一轮体验不同（前世天赋随机）
- **凡人修仙传世界观**：所有命名/设定致敬《凡人修仙传》，角色"人→鼠"化

### 1.3 核心循环

```
打怪 → 经验 + 灵石 + 装备掉落 + 材料 + 天机令
  ↓
灵石 → 强化装备 / 炼丹 / 喂灵兽 / 洞府 / 功法升级
材料 → 炼丹
天机令 → 天机阁抽取外观
  ↓
变强 → 更高秘境 / 更高塔层 / 渡劫突破
  ↓
新境界 → 解锁新功法 + 新秘境 + 新怪物 + 新灵兽 + 新外观
  ↓
大乘期(Lv.50+) → 🌟飞升转生 → 仙缘点 → 永久加成 + 前世天赋
  ↓
新一世修仙（更强更快！）→ 循环
```

## 3. 战斗系统

### 3.1 基础战斗
- **自动攻击**：每 tick（默认2秒）鼠鼠自动攻击当前怪物
- **伤害公式**：`attack × (isCrit ? critDamage/100 : 1) × random(0.9~1.1)`
- **防御减伤**：`max(1, monsterAtk - defense × 0.3)`
- **暴击**：根据暴击率触发，伤害乘以暴击伤害百分比
- **闪避**：双方均有闪避率，闪避后本次攻击无效

### 3.2 战斗速度
三档可调：
- 1x = 2秒/tick（默认）
- 2x = 1秒/tick
- 4x = 0.5秒/tick

### 3.3 生命系统
- **自然回复**：每tick回复1%最大生命（HP<30%时额外+0.5%）
- **死亡**：HP归零后鼠鼠死亡，5秒后复活（渡劫失败15秒）
- **死亡惩罚**：掉落10%灵石
- **自动吃药**：可配置血量阈值（20%~60%），低于阈值自动使用回元丹

### 3.4 精英怪
- **触发**：连续击杀15次(±3随机)后出现
- **特征**：多管血条（2-4管）、金色光环、名字金色显示
- **掉落翻倍**：经验×2、灵石×2
- **天机令**：50%概率掉落天机令

### 3.5 怪物特性（9种）
| 特性 | 效果 |
|---|---|
| poison（毒） | 每回合2%最大生命持续伤害，3回合 |
| dodge（闪避） | 怪物闪避率提高 |
| thorns（荆棘） | 反弹受到伤害的一部分 |
| berserk（狂暴） | 攻击力提高 |
| slow（减速） | 降低玩家攻速 |
| burn（灼烧） | 每回合3%最大生命持续伤害，3回合 |
| critBoost（会心） | 怪物暴击率提高 |
| lifesteal（吸血） | 攻击回复8%伤害值的HP（HP>0时才触发） |
| charm（魅惑） | 概率使鼠鼠攻击自己 |

### 3.6 DPS统计
- 实时计算每秒伤害量
- 滑动窗口平均（最近10次攻击）

## 5. 装备系统

### 5.1 槽位
4个装备槽：武器、衣服、饰品、鞋子

### 5.2 品质体系
| 品质 | 颜色 | 掉率 | 词条数 |
|---|---|---|---|
| 白（凡品） | #CCCCCC | 40% | 0 |
| 绿（良品） | #44CC44 | 30% | 1 |
| 蓝（稀有） | #4488FF | 18% | 2 |
| 紫（珍品） | #AA55FF | 8% | 3 |
| 橙（极品） | #FF8800 | 3.3% | 4 |
| 红（传说） | #FF2222 | 0.7% | 5 |

### 5.3 词条系统（10种）
攻击、防御、生命、暴击率、暴击伤害、吸血、闪避、攻速、经验加成、灵石加成

### 5.4 装备强化
- 消耗灵石，最高+15
- 费用公式：`100 × 1.8^(强化等级) × (品质索引+1)`
- 强化加成叠加到基础属性上

### 5.5 装备掉落
- 基础掉率：5%
- 境界加成：每个境界+1%
- 精英怪掉率翻倍

### 5.6 背包
- 30格上限
- 装备对比：自动显示▲▼=标记
- 一键换装：自动穿最强装备
- 一键卖弱装：出售比身上弱的装备

### 8.2 灵兽机制

- **灵兽伤害**：`(baseAtk + level×2) × random(0.8~1.2)`
- **一键喂满**：一次花完灵石喂到上限

## 11. 锁妖塔

- **无尽逐层挑战**，模拟回合制战斗
- 怪物随层数缩放（每10层切换怪物档次）
- **通关奖励**：经验 + 灵石
- **记录最高层数**
- **里程碑**：每10层/25层/50层/100层有额外奖励
- **扫荡**：可一键扫荡到最高层（消耗时间）
- **连续闯**：自动连续挑战直到失败

## 12. 渡劫系统

- **触发**：达到大境界上限（Lv.9→10, 19→20, 29→30...）
- **操作**：手动点击"渡劫"按钮
- **基础成功率**：60%
- **加成**：
  - 装备战力加成：上限15%
  - 金元丹：+20%
  - 灵兽：+3%
- **成功率上限**：95%
- **失败**：15秒冷却，不掉级
- **成功**：大幅属性提升 + 10修炼点

## 18. 数值体系

### 18.1 升级
- 经验公式：`50 × 1.35^(level-1)`
- 升级属性增长：攻击×1.12+2、防御×1.08+1、生命×1.1+10

### 18.2 掉落
- 材料掉率：35%
- 装备掉率：5% + 境界×1%
- 奇遇触发率：8%（30秒CD）

### 18.3 灵石消耗去向
- 装备强化
- 功法升级
- 灵兽喂养
- 洞府建设
- 炼丹
- 抽卡（通过天机令间接消耗战斗时间）

## 19. 视觉系统

### 19.1 渲染方式
- **v1.x**: Canvas 2D `fillRect()` 逐像素手绘
- **v2.0**: Sprite Sheet PNG 图集 + `drawImage()` 裁切渲染
  - 图集规格：48×48（怪物/灵兽/鼠鼠）、64×48（坐骑）、32×32（武器）
  - 每个精灵4帧动画
  - 离屏Canvas缓存

### 19.2 战斗动画
- 鼠鼠冲刺攻击（10帧）
- 怪物受击闪白+震动（8帧）
- 灵兽冲刺攻击+能量弹道（15帧周期）
- 功法释放特效（25%触发，10种视觉类型）
- 击杀爆炸粒子
- 境界突破白屏闪+金色粒子爆炸

### 19.3 伤害数字
- 重力弹跳物理（2次反弹）
- 暴击金色大字，放大效果
- 防重叠自动偏移

### 19.4 环境系统
- 日夜循环（太阳/月亮/星星/云彩）
- 场景小景（像素树×3、岩石×2、草地摆动）
- 境界环境粒子（6种风格）
- 鼠鼠呼吸动画（上下浮动+左右摆动）

### 19.5 连杀特效
- 10/25/50/100连杀里程碑大字
- >=10连杀屏幕边缘发光（三级颜色）

## 23. 技术架构（v2.0 重构后目标）

```
public/
├── index.html         — HTML 结构 + CSS 样式（纯标记，无逻辑）
├── engine.js          — 游戏引擎（状态管理 + 全部游戏逻辑）
├── renderer.js        — Canvas 渲染系统（场景/动画/粒子/特效）
├── ui.js              — DOM UI 更新（标签页渲染/状态面板/事件绑定）
├── main.js            — 初始化入口（启动引擎/绑定事件/离线处理）
├── sprites.js         — 像素精灵（Canvas fallback 绘制，过渡期保留）
├── sprite-loader.js   — Sprite Sheet 加载系统（PNG图集→drawImage）
└── assets/            — 精灵图集 PNG 文件
    ├── monsters.png   — 18种怪物（48×48, 4帧×18行）
    ├── beasts.png     — 6种灵兽（48×48, 4帧×6行）
    ├── mounts.png     — 2种坐骑（64×48, 4帧×2行）
    ├── mouse.png      — 鼠鼠（48×48, 6境界×2状态×4帧）
    └── weapons.png    — 武器皮肤（32×32, 20种）
```
